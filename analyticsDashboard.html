⚠️ Important: Activate Email Delivery (one-time setup)

The email system uses Formsubmit.co (free, no backend). The FIRST time an email is sent to each admin address, Formsubmit holds it and sends an activation link. Until both admins click that link, no emails arrive. This is a one-time setup — do it once, emails flow forever after.

Steps to activate:
1. Click "Send Test Email" above (it triggers one email to each admin)
2. Both admins (Salon & Tsering) check their Gmail inbox AND spam folder for an email from noreply@formsubmit.co
3. Click the blue "Activate" button in that email
4. Done! All future order/inquiry emails will arrive automatically

After you complete this, place a test order — you should receive the order email within 15 seconds.

How this works: When an order is placed or inquiry submitted, both admins (Salon & Tsering) automatically receive an email. For WhatsApp, tap the button next to each notification to send the pre-filled message via WhatsApp (browsers cannot send WhatsApp silently — this is intentional, to prevent spam).

How It Works

1. All store data syncs automatically via Firebase Realtime Database — no tokens or manual setup needed.

2. When a customer submits a form or places an order on any device, it appears here instantly.

3. Firebase listeners push updates to all connected browsers in real-time.

4. Cart data stays local to each browser (not synced) — this is by design.

✓ Free tier: 1 GB storage, 10 GB/month transfer — more than enough for this project.
