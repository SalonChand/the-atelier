How It Works

1. All store data syncs automatically via Firebase Realtime Database — no tokens or manual setup needed.

2. When a customer submits a form or places an order on any device, it appears here instantly.

3. Firebase listeners push updates to all connected browsers in real-time.

4. Cart data stays local to each browser (not synced) — this is by design.

✓ Free tier: 1 GB storage, 10 GB/month transfer — more than enough for this project.
